list_t Struct Reference

#include <list.h>

Detailed Description

This is the linked list object itself. You want to embed this in your parent structures which contain the linked list, or dynamically allocate it.

Before using the linked list, it must be initialized using list_create(). After you are done with the list, you must deinitialize it using list_destroy(). Items in the list can be added using the list_insert_* and list_remove_* functions. The list can be traversed using list_head(), list_tail(), list_next() and list_prev().
