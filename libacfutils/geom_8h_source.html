384 /*
385  * ECmI stands for Earth-Centered-modified-Inertial. Unlike ECEF, it is
386  * an inertial frame that doesn't rotate together with the earth. However,
387  * unlike plain ECI coordinates, ECmI is aligned with the Earth equatorial
388  * plane, not the Earth's orbital plane around the Sun. This makes it a
389  * bit easier to use for inertial calculations for objects moving around
390  * on the Earth (like airplanes, duh). In ECmI, the X and Y axes are
391  * aligned with the 0 and 90 degree meridians respectively ONLY at the
392  * coordinate reference time (delta_t=0). Thus, to convert between
393  * geographic or ECEF coordinates and ECmI, we need to know the exact time
394  * as a delta from the reference time (which can be arbitrary).
395  */
