357 /*
358  * ECmI stands for Earth-Centered-modified-Inertial. Unlike ECEF, it is
359  * an inertial frame that doesn't rotate together with the earth. However,
360  * unlike plain ECI coordinates, ECmI is aligned with the Earth equatorial
361  * plane, not the Earth's orbital plane around the Sun. This makes it a
362  * bit easier to use for inertial calculations for objects moving around
363  * on the Earth (like airplanes, duh). In ECmI, the X and Y axes are
364  * aligned with the 0 and 90 degree meridians respectively ONLY at the
365  * coordinate reference time (delta_t=0). Thus, to convert between
366  * geographic or ECEF coordinates and ECmI, we need to know the exact time
367  * as a delta from the reference time (which can be arbitrary).
368  */
